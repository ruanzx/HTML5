Excel Import and Export sample

This sample shows how to import Excel file to SpreadJS widget and export Excel file from SpreadJS widget.

All samples depend on the Excel Import and Export service that resides on your application server. 
The Excel Import and Export service requires IIS 7.0 or higher with ASP.NET. 
ASP.NET 4.0 is required with Microsoft Windows 7 and ASP.NET 4.5 is required with Microsoft Windows 8. 
The 4.0 .NET framework is also required. 
The default Excel IO host is http://localhost/ExcelIO/ after installing the Excel IO setup.
And there is only two API service exposed: 
  Excel Import API: http://localhost/ExcelIO/xsapi/import. 
  Excel Export API: http://localhost/ExcelIO/xsapi/export.
