Web Installer Instructions

To install Spread Studio,

  1. Run the SpreadInstaller.exe program.
  2. Choose the product (Assemblies) and documentation (Documentation) you want to install.
  3. If you want to change the installation path, type the path in the Install Path box or click the ellipses button to browse to specify the path.
  4. Check the Accept EULA box to indicate that you agree to the terms of the license agreement.
  5. Click Install to begin downloading and installing the selected Spread platform components and documentation.

Note: The web installer requires .NET Framework 3.5 or above.

Instructions

To install SpreadJS,

  1. Save the downloaded ZIP file (SpreadJSxxxx.zip) to a temporary directory on your system, and then unzip the files to a directory.

The default SpreadJS folders:

Install Directory File Description
\SpreadJS.xxxx   Main directory for product
\SpreadJS.xxxx\css   CSS files
\SpreadJS.xxxx\samples   Sample pages for the widget
\SpreadJS.xxxx\scripts gcspread.sheets.xxxx.min Minimized js file for widget which only includes basic functions
  gcspread.sheets.functions.xxxx.min Minimized js file for extension functions
  gcspread.sheets.all.xxxx.min Minimized js file for widget which includes all functions
Additional js files Foundation support files
\SpreadJS.xxxx\definition gcspread.sheets.d.ts SpreadJS TypeScript API definition file
\SpreadJS.xxxx\scripts\pluggable   Pluggable js files

To install the SpreadJS designer,

  1. Run the setup (SpreadJS Sheets Designer.exe). Unlock advanced features such as the Excel Import and Export service with the designer license.

The default SpreadJS designer folder:

Install Directory Description
C:\ProgramData\GrapeCity\Spread Studio 8\SpreadJS\Designer Designer directory for product

To install the Excel Import and Export service,

  1. Run the setup (SpreadJS ExcelIO Server Component.exe).

The default folder:

Install Directory Description
C:\ProgramData\GrapeCity\Spread Studio 8\SpreadJS\ExcelIO Excel Import and Export service for product

Refer to Upgrading the Excel Import and Export Service for upgrade steps.

The Excel Import and Export service requires IIS 7.0 or higher with ASP.NET. ASP.NET 4.0 is required with Microsoft Windows 7 and ASP.NET 4.5 is required with Microsoft Windows 8. The 4.0 .NET framework is also required.
