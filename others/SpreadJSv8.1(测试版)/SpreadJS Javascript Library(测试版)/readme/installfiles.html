Installed Files
ReadMe > Installed Files

By default, SpreadJS installs the following files on your system:

Install Directory File Description
\Program Files (x86)\ComponentOne - Company directory for installed products
\Program Files\ComponentOne\SpreadJS - Directory for files common to GrapeCity products

SpreadJS files:

Install Directory File Description
\Program Files (x86)\ComponentOne\SpreadJS\1.0.0.0\Client - Main directory for product
\Program Files\ComponentOne\SpreadJS\1.0.0.0\Client\scripts - Script files for client control
jquery.wijmo.wijspread.min Minimized js file for control which only includes basic functions
jquery.wijmo.wijspread.functions.min Minimized js file for extension functions
jquery.wijmo.wijspread.full.min Minimized js file for control which includes all functions
\Program Files\ComponentOne\SpreadJS\1.0.0.0\Client\css Style sheets for the control
\Program Files\ComponentOne\SpreadJS\1.0.0.0\Client\samples Sample pages for the control
